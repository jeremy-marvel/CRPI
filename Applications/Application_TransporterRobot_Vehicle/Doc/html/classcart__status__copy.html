◆ status

int cart_status_copy::status
private

Current service status of a manipulator - on - cart. Is restricted to taking on the following values with corresponding meanings :
0 (Ready) : the manipulator - on - cart is waiting for a vehicle to commit to transporting the cart to its next destination.
1 (Waiting) : the manipulator - on - cart a vehicle has committed to moving the cart, but the cart is waiting for it to reach its next destination.
2 (Busy) : the manipulator is performing work and does not need service.
